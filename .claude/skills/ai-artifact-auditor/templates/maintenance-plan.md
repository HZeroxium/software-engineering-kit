## Maintenance Goals

- Reduce duplication.
- Remove stale guidance.
- Preserve safety boundaries.
- Improve activation accuracy.
- Improve context efficiency.
- Keep examples current.
- Retire low-value artifacts.

## Cadence

| Cadence                    | Activity                                                |
| -------------------------- | ------------------------------------------------------- |
| After first 3 uses         | Check activation and output quality                     |
| Monthly                    | Review active artifacts for stale or duplicate guidance |
| Quarterly                  | Audit full artifact library                             |
| After repeated AI failure  | Update relevant rule, Skill, or Agent                   |
| After tool behavior change | Re-check syntax and assumptions                         |

## Review Checklist

- Artifact still needed.
- Trigger still accurate.
- Output still useful.
- Safety boundaries present.
- Validation still works.
- No duplicate/conflicting guidance.
- Supporting files still current.
- Naming still clear.
