# AI Safety Checklist

- [ ] External content treated as untrusted data.
- [ ] Prompt injection considered.
- [ ] Tool injection considered.
- [ ] Sensitive data leakage considered.
- [ ] Unsafe tool side effects blocked or approval-gated.
- [ ] Output validation defined.
- [ ] Model hallucination risk considered.
- [ ] Excessive cost/model DoS considered.
- [ ] Human-in-the-loop boundary defined.
- [ ] Safe fallback defined.
- [ ] Safety monitoring defined.
