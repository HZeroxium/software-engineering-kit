# Privacy Checklist

- [ ] Sensitive data classified.
- [ ] Data minimization considered.
- [ ] Data retention considered.
- [ ] Data deletion considered.
- [ ] Logs reviewed for sensitive data.
- [ ] Metrics/traces reviewed for sensitive data.
- [ ] Events/messages reviewed for sensitive data.
- [ ] Model/provider context reviewed.
- [ ] External sharing reviewed.
- [ ] Redaction/masking/tokenization considered.
- [ ] Privacy/security review requested where needed.
