# Logging Checklist

- [ ] Important events logged.
- [ ] Log levels appropriate.
- [ ] Correlation/request ID included if available.
- [ ] Outcome included.
- [ ] Error category included.
- [ ] No secrets.
- [ ] No sensitive payloads.
- [ ] No unbounded large payloads.
- [ ] Business errors not logged as system failures.
- [ ] Logs are useful for diagnosis.
