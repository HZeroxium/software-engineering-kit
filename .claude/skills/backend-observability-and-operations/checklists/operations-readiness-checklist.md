# Operations Readiness Checklist

- [ ] Build/validation command known.
- [ ] Config changes documented.
- [ ] Health checks reviewed.
- [ ] Logs/metrics/traces ready.
- [ ] Alerts/runbooks ready where needed.
- [ ] Rollback path known.
- [ ] Feature flag/kill switch considered.
- [ ] Migration compatibility checked.
- [ ] Sensitive data risk checked.
- [ ] Owner and escalation path known.
