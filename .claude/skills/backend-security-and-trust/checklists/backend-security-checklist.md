# Backend Security Checklist

- [ ] Assets identified.
- [ ] Actors identified.
- [ ] Trust boundaries identified.
- [ ] Authentication checked.
- [ ] Authorization checked.
- [ ] Object-level authorization checked.
- [ ] Tenant isolation checked.
- [ ] Sensitive data classified.
- [ ] Secrets handling checked.
- [ ] Input validation checked.
- [ ] Output exposure checked.
- [ ] External calls checked.
- [ ] Abuse/rate limit risk checked.
- [ ] Audit logging checked.
- [ ] Security tests or manual review planned.
