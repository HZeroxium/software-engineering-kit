# Privacy and Sensitive Data Checklist

- [ ] Sensitive data types identified.
- [ ] Data minimization considered.
- [ ] API response fields reviewed.
- [ ] Logs reviewed for sensitive values.
- [ ] Metrics/traces reviewed for sensitive labels/attributes.
- [ ] Events/messages reviewed.
- [ ] Cache reviewed.
- [ ] External provider sharing reviewed.
- [ ] Retention/deletion considered.
- [ ] Masking/redaction strategy exists.
- [ ] Manual privacy/security review requested if needed.
