# Database Performance Checklist

- [ ] Query result size bounded.
- [ ] Filters are selective.
- [ ] Sorts are deterministic.
- [ ] Index supports filters/sorts.
- [ ] Pagination is efficient.
- [ ] N+1 query risk checked.
- [ ] Transaction duration checked.
- [ ] Lock contention checked.
- [ ] Connection pool usage checked.
- [ ] Query plan reviewed when needed.
