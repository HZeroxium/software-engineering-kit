# Performance Review Checklist

- [ ] Performance target defined.
- [ ] Measurement evidence collected or missing evidence stated.
- [ ] p95/p99 considered.
- [ ] Throughput considered.
- [ ] Database queries reviewed.
- [ ] External calls reviewed.
- [ ] Cache behavior reviewed.
- [ ] Concurrency/resource pools reviewed.
- [ ] Payload size reviewed.
- [ ] Cost impact considered.
- [ ] Validation plan defined.
