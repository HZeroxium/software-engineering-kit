# Incident Readiness Checklist

- [ ] User-impacting alerts exist or are planned.
- [ ] Dashboard exists or is planned.
- [ ] Logs/traces are diagnosable.
- [ ] Runbook exists or is planned.
- [ ] Rollback path exists.
- [ ] Escalation path is clear.
- [ ] Owner is clear.
- [ ] Known failure scenarios listed.
- [ ] Mitigation steps are safe.
- [ ] Postmortem/action-item process exists or is planned.
