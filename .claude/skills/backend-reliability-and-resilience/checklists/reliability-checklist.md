# Reliability Checklist

- [ ] Critical user journey identified.
- [ ] User-visible failure defined.
- [ ] Dependencies mapped.
- [ ] Timeouts defined.
- [ ] Retry safety checked.
- [ ] Fallback/degradation defined.
- [ ] Circuit breaker need considered.
- [ ] Bulkhead/resource isolation considered.
- [ ] Backpressure considered.
- [ ] SLO/SLI considered.
- [ ] Alerts/runbooks considered.
- [ ] Resilience tests planned.
