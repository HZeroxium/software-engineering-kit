# Dependency Failure Checklist

- [ ] Dependency role identified.
- [ ] Timeout failure considered.
- [ ] Error response considered.
- [ ] Slow response considered.
- [ ] Rate limit considered.
- [ ] Partial success considered.
- [ ] Stale data considered.
- [ ] Dependency outage considered.
- [ ] Retry behavior defined.
- [ ] Fallback behavior defined.
- [ ] Observability defined.
